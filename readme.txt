安装之前准备工作：把你的PHP环境配置好，程序只支持php5.3+，mysql5.1+

安装教程：

把主程序解压出来，以二进制的方式上传到站点里，在地址栏输入您的站点，

安装完成后，进入后台-平台-网站缓存 一下即可。

安装完后，为了安全考虑，本程序做了目录调整，请把根目录/install目录下的lock，除了lock文件，其它的请删除。

如果安装后，打开界面上比较乱，说明安装不成功，请把 根目录/install目录下的lock 文件删除，

请检查安装频道里的环境支持及目录及子目录有写入、修改权限，再重新安装。

程序源码为uft-8，所以不要用记事本编辑程序中的每个文件，推荐Notepad++文本编辑器。
